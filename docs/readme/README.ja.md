Orca: ALab Edition は、[Stably の Orca](https://github.com/stablyai/orca)
を基にした実験的な下流フォークで、パフォーマンスと正確性を重視しています。
[github.com/andrewyatesai/orc](https://github.com/andrewyatesai/orc)
でメンテナンスされており、ターミナルスタック、ネイティブのホットパス、
障害からの復旧に注力しています。

> **注意:** アップストリームのダウンロードでインストールされるのは
> アップストリーム版の Orca です。Orca: ALab Edition は現在、
> このリポジトリからビルドされています。

このフォークの完全なドキュメントは英語の [README](../../README.md) と
[機能ウォークスルー](../../FEATURE_WALKTHROUGH.md)にあります。アップストリームの
完全な製品ガイドは[アップストリームのリポジトリ](https://github.com/stablyai/orca)
と <https://www.onorca.dev/docs> を参照してください。
