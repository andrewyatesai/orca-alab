Orca: ALab Edition est un fork dérivé et expérimental de
[l'Orca de Stably](https://github.com/stablyai/orca), axé sur les performances
et la fiabilité. Il est maintenu sur
[github.com/andrewyatesai/orca-alab](https://github.com/andrewyatesai/orca-alab) et se
concentre sur la pile terminal, les chemins critiques natifs et la reprise
après incident.

> **Avertissement :** les téléchargements du projet d'origine (upstream)
> installent l'Orca d'origine. Orca: ALab Edition est actuellement compilé à
> partir de ce dépôt.

La documentation complète de ce fork se trouve dans le
[README](../../README.md) en anglais et dans la
[présentation des fonctionnalités](../../FEATURE_WALKTHROUGH.md) ; le guide
produit complet du projet d'origine est disponible dans le
[dépôt upstream](https://github.com/stablyai/orca) et sur
<https://www.onorca.dev/docs>.
